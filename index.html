Kendi Web Sitenizi Yapın!

1. Adım:
PC'den site kurmanız daha iyi olur çünkü kodlama için PC gerekir. (Ben de PC'den yapıyorum.)

2. Adım:
Siteyi oluşturmak için GitHub Pages kullanacağız. İlk olarak bir GitHub hesabı açın.

GitHub Pages, sitenizi barındırmak için ücretsizdir.

3. Adım:
Yeni bir repository oluşturun ve aşağıdaki kodları bir dosyaya kaydedin:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>My GitHub Page</title>
</head>
<body>
  <h1>Merhaba, Benim Sitem!</h1>
</body>
</html>

4. Adım:
Dosyayı "index.html" olarak kaydedin ve GitHub'daki repository'ye yükleyin. "Settings" sekmesinden Pages kısmını etkinleştirin.

Bu, temel bir site oluşturur. Daha gelişmiş bir site için daha fazla kod yazmalısınız!

Python'un Temelleri

1. Adım: Değişkenler ve Veri Tipleri
Python'da değişkenler, değerleri saklamak için kullanılır. Veri tipleri ise değişkenlerin alabileceği değer türlerini belirtir. Örnek:

x = 5
y = "Merhaba"
print(x)
print(y)

2. Adım: Fonksiyonlar
Python'da fonksiyonlar, tekrar kullanılabilir kod bloklarıdır. Örnek:

def selamla():
  print("Merhaba!")
selamla()

3. Adım: Koşul İfadeleri
Koşullu ifadeler, bir koşulun doğru olup olmadığını kontrol eder. Örnek:

sayi = 10
if sayi > 5:
  print("Sayı 5'ten büyük")

4. Adım: Döngüler
Döngüler, belirli bir kodu birden fazla kez çalıştırmak için kullanılır. Örnek:

for i in range(5):
  print(i)

5. Adım: Listeler
Listeler, sıralı veri koleksiyonlarıdır. Örnek:

meyveler = ["elma", "armut", "muz"]
print(meyveler[0])

6. Adım: Sözlükler
Sözlükler, anahtar-değer çiftleriyle veri depolar. Örnek:

kisi = {"isim": "Ahmet", "yas": 25}
print(kisi["isim"])

7. Adım: Hata Yakalama (try-except)
Hata yakalama, programın hatalarla karşılaştığında çökmesini engeller. Örnek:

try:
  sonuc = 10 / 0
except ZeroDivisionError:
  print("Sıfıra bölme hatası!")
